需要关注的一些 weex 样式

发表于 2017-11-13 | 分类于 weex

需要关注的一些 weex 样式

  1. Weex 盒模型基于 CSS 盒模型 。box-sizing 默认为 border-box，即盒子的宽高包含内容、内边距和边框的宽度，不包含外边距的宽度。
  2. Weex 对于长度值目前只支持 px 长度单位，不支持相对单位（em、rem、pt），暂时不支持百分值％;
  3. weex 的样式不支持简写，譬如：border 目前不支持类似这样border: 1 solid #ff0000;的组合写法。
  4. Weex 目前不支持 z-index 设置元素层级关系，但靠后的元素层级更高，因此，对于层级高的元素，可将其排列在后面。

svg基础动画

发表于 2016-01-08 | 分类于 css3动画

主要通过学习svg的基础属性，实现一些很基础的动画。

  • attributeName: 动画属性名，需改变的属性
  • attributeType: 动画属性的类别，常见的两个值，XML和css，取决于属性是在xml里还是在style里
  • from，to: 开始和结束值，from可选，当不写的时候会取默认值
  • begin，dur: 动画的开始时间与结束时间
  • fill: 参数有freeze和remove. 其中remove是默认值，表示动画结束直接回到开始的地方
  • repeatCount：循环次数。取值为数字或indefinite，indefinite表示无限循环
